# Object Types / Data Types

- Number : 1234, 3.1415, 4+5j, 0b1010, 0o123, 0x123, Decimal('0.1'), Fraction(1, 3)
- String : 'spam', "Bob's", r'\temp', """long string""", u'sp\xc4m', b'a\x01c'
- List : [1, [2, 'three'], 4.5], list(range(5))
- Tuple : (1, [2, 'three'], 4.5), tuple(range(5)), namedtuple
- Dictionary : {'food': 'spam', 'taste': 'yum'}, dict(a=1, b=2, c=3), defaultdict(list), OrderedDict
- Set : set('abc'), {1, 2, 3}, set(range(5)), frozenset('spam')
- File : open('data.txt'), io.StringIO('in memory text stream')
- Boolean : True, False
- None : None
- Functions, modules, classes
- Advance: Decorators, generators, iterators, MetaProgramming, etc.
